Positioning of Elements with CSS Property "position"

Static HTML Flow:

Im folgenden werde ich verschiedene Beispiele wie oben zu sehen zeigen und erläutern. Das obere Beispiel besteht aus den Elementen:

  • h1
  • 3 divs für die farbigen Hintergründe, vor denen die farbigen Formen positioniert sind
  • divs für die farbigen Formen

Dabei wurden sowohl die unteren beiden Hintergründe als auch die farbigen Formen auf display: inline-block; gesetzt.
Anhand der Sortierung im oberen Hintergrund erkennt man, dass sich inline-block Elemente mit unterschiedlicher Höhe standardmäßig so positionieren, dass die unteren Kanten auf einer Höhe liegen.
Anhand der Sortierung in den unteren beiden Hintergründen, die auch beide inline-block Elemente sind, erkennt man, dass sich die vertikale Positionierung eines Parent Elements im inline-block Modus mit Child Elementen im inline-block Modus so ausrichtet, dass die oben beschriebene Sortierung der Child Elemente gegeben ist. Dadurch ist der rechte Hintergrund nach unten verrutscht.
